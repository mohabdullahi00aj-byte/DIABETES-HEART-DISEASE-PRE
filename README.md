# Segmentation & Prediction of Diabetes Risk Using Machine Learning

## Overview
This project applies both supervised and unsupervised machine learning techniques
to address healthcare challenges related to diabetes and heart disease.

## Notebooks
- `Supervised ML.ipynb` – Diabetes risk prediction using LR, DT, RF, SVM
- `Unsupervised ML.ipynb` – Heart disease patient segmentation using K-Means

## Techniques Used
- K-Means Clustering
- Logistic Regression
- Decision Tree
- Random Forest
- Support Vector Machine (SVM)
- PCA & SHAP Explainability

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score
- ROC-AUC
